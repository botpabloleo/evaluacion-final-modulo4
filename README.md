# Evaluación Final Módulo 4

Aplicación web que consume el endpoint `https://jsonplaceholder.typicode.com/users` mediante `XMLHttpRequest` para administrar y mostrar información de usuarios.

## Objetivos

- Consumir un web service con `XMLHttpRequest`
- Administrar la data dentro de una clase JavaScript
- Listar, buscar y mostrar información de usuarios por consola y en pantalla
